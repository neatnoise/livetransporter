# livetransporter
LiveTransporter - application to help transfer livestreams to youtube platform

This is an application project which could help streamers to archive livestreams on YT.
Currently supported livestream services: twitch.tv, hitbox.tv

This project uses components of other open applications:
- Youtube-upload: https://github.com/tokland/youtube-upload
- Livestreamer: https://github.com/chrippa/livestreamer

Supported python version: 2.7
